Voice Classificator

•Запуск:
	1. Для запсука со стандартной конфигурацией достаточно указать только директорию: ./vc --dir=INPUT_DIR
	2. Опц. параметры: 
		--single_file_mode='on' - для обработки одного файла (по умолчанию выключен)
		--wav_path - если включен single_file_mode, то необходимо указать путь к файлу (по умолчанию None)
		--output - директория, в которой будут сохраняться результаты (по умолчанию 'output')
		--weights - файл с обученной нейросетью (по умолчанию 'weights.hdf5')
		--fragments - количество фрагментов, которые будут вырезаны из файла (по умолчанию 5, то есть классификация будет проведена на первых 5 секундах речи)
		--volume_threshold - порог громкости для удаления фрагментов без голоса (по умолчанию 10000)
		--loop_timer - как часто директория проверяется на наличие новых файлов (по умолчанию 1 (в секундах))

•Завершение:
	Выход из скрипта CTRL + C

•Результат работы:
	На выходе json файл для каждого .wav файла следующего вида:
